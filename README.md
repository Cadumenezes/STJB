# 💃 DanceFlow - Dance School Management System

DanceFlow is a comprehensive, modern management platform designed specifically for dance studios. It streamlines operations, manages students, tracks attendance, and handles finances with a premium, user-friendly interface.

## ✨ Features

- **Dashboard**: Real-time overview of studio activities and metrics.
- **Student Management**: Complete database of students with registration and status tracking.
- **Attendance Control**: Easy tracking of 'present', 'delayed', or 'absent' status.
- **Financial Module**: Automated payment workflows, monthly fee generation, and cash flow monitoring.
- **Inventory & Team**: Manage school supplies and team members effortlessly.
- **Customizable UI**: Personalize the look and feel to match your studio's branding.

## 🚀 Technologies

- **Frontend**: React + TypeScript + Vite
- **Styling**: Vanilla CSS (Modern Design System)
- **Database/Backend**: Supabase
- **Icons**: Lucide React

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Cadumenezes/danceflow.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Variables**:
   Create a `.env` file in the root and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
